# Flavor Flaves
A Recipe Manager app built using Rails.

## Installation
Fork the repo.
Run `bundle install` to install necessary gems.
Run 'rake db:migrate' to create the database.
A small seed file is included.  Run 'rake db:seed' to populate the database.
